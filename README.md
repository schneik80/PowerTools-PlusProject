# PowerTools-PlusProject
Access additional tools in your fusion projects to track tasks and work. Add-in allows linking Fusion projects to external project management services, specificaly this addin is branded for ClickUp.

Two commands are provided.
Save URL - When a document is open, you may save to the cache a link to your a specific page in Clickup.
Open - When any document in a project is open, you can open the project specific project managment page.

Example usage.

For each Fusion project you create a peer project in ClickUp. Your create a task list to track project work. Saving this URL to the Cache allows users to quickly open the task list when working in Fusion.

You can have unique task lists for each fusion project.
